# Test Ride Booking

A modern web application built with Next.js for booking test rides. This project features a responsive design with sections for hero, problem statement, solutions, user segments, metrics dashboard, roadmap, and more.

## Tech Stack

- **Framework**: Next.js 14
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Deployment**: Ready for Vercel or any Node.js hosting

## Features

- Responsive landing page with multiple sections
- Animated counters and metrics dashboard
- Conversion funnel visualization
- User segment analysis
- Product solutions showcase
- Roadmap and final CTA sections

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Saurya29/TestRideBooking.git
   cd TestRideBooking
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   ```

3. Run the development server:
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to see the application.
